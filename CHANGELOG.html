🎨 Frontend İyileştirmeleri
  • 2 yeni modal (Sipariş Geçmişi, Masa Notu)
  • Arama kutusu dinamik filtreleme
  • Favori butonları toggle animasyonu
  • Masa kartlarında not ikonu göstergesi
📊 API Endpoints (Yeni)
  • PATCH /api/categories/<id>/order - Kategori sıralaması
  • PATCH /api/products/<id>/order - Ürün sıralaması
  • PATCH /api/products/<id>/favorite - Favori toggle
  • GET /api/products/search?q=... - Ürün arama
  • GET /api/orders/history?date=... - Sipariş geçmişi
  • POST /api/orders/<id>/reopen - Sipariş yeniden açma
  • PATCH /api/tables/<id>/note - Masa notu güncelleme
  • GET/PUT /api/settings/<key> - Ayar yönetimi
